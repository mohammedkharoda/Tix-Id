An App that help you to book ticket for your favourite movie without too much hustle !!


## Run the Project

To run this project :

- Clone the Project
```bash
  git clone https://github.com/mohammedkharoda/Tix-Id.git
```

- Install the node modules
```bash
  npm install
```
- Let's Go!!
```bash
  npm run start
```



## Tech Stack

**Client:** React, Redux , React Router

**Styling:** MUI library
